
# Roadmap

## Done (MVP)

- Home Assistant config flow
- Persistent cat registry and last event storage
- Service API for event ingestion and cat management
- Basic sensors and binary sensors
- ESPHome simulation buttons for integration testing

## Next

1. Add diagnostics and logging entities (RSSI, uptime, error counters).
2. Add direction detection pipeline from front/back ToF sensors on ESPHome.
3. Add real RFID reader support for 134.2 kHz chip readers.
4. Add Home Assistant options flow for managing cats in UI.
5. Add tests for service handlers and entity state transitions.
